# Product Overview

HarekazeScan（晴れ風API）は、桜の木の健康状態を診断・管理するためのバックエンドAPIサービスです。一般ユーザーが桜の写真を投稿し、AIを活用した樹勢診断や樹齢推定を行います。

## Core Capabilities

- **樹勢診断**: 桜の木全体の写真からAIで元気度（1-5段階）を判定
- **樹齢推定**: 幹の写真と缶（スケール参照）から周囲長・テクスチャを解析し樹齢を推定
- **健康状態記録**: 幹の穴、テングス病、キノコ、コブなど異常状態を記録
- **地域統計**: 都道府県・市区町村単位での桜の分布・健康状態の集計
- **コンテンツモデレーション**: 事務局による検閲ワークフロー（承認/NG/エスカレーション）

## Target Use Cases

- 一般市民による桜の木の撮影・投稿
- 地域ごとの桜の木の健康状態可視化
- 桜の開花日予測データとの連携
- 事務局による投稿内容の品質管理

## Value Proposition

- AWS Rekognition と Lambda を活用したAI解析パイプライン
- 位置情報ベースの地域検索（GeoAlchemy2による空間クエリ）
- プライバシー保護のための人物自動ぼかし機能
- モバイルアプリ連携を想定したRESTful API設計
